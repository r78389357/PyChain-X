# PyChain-X: 下一代高性能 Python 区块链框架

PyChain-X 是一个从零开始构建的、模块化的、支持智能合约的区块链底层框架。它不依赖第三方库，纯原生 Python 实现，旨在展示分布式账本技术的核心原理与高级架构。

## 🚀 项目亮点

- **纯原生实现**：不依赖重型框架，深入理解底层字节流与加密算法。
- **模块化架构**：将共识、网络、账本、钱包完全解耦。
- **安全性**：内置椭圆曲线加密、默克尔树验证、工作量证明机制。
- **高性能**：优化的链结构设计与内存池管理。

## 📂 核心文件清单与功能

本项目覆盖从底层密码学到上层应用接口的全栈开发：

### 核心层
- `genesis.py`: 创世区块硬编码与初始化。
- `block.py`: 区块数据结构定义（包含时间戳、哈希、父哈希）。
- `transaction.py`: 交易模型，包含输入输出与签名验证。
- `chain.py`: 区块链主类，负责链的增删改查与验证。
- `merkle.py`: 默克尔树算法实现，用于交易指纹验证。

### 共识与挖矿
- `pow.py`: 工作量证明算法，动态难度调整。
- `consensus.py`: 共识接口与策略模式实现。
- `miner.py`: 挖矿进程管理，监听新区块与打包交易。

### 网络层
- `p2p.py`: P2P 网络节点发现与握手协议。
- `network.py`: 广播机制，负责交易与区块的传播。
- `protocol.py`: 自定义二进制通信协议解析。

### 账户与钱包
- `wallet.py`: 钱包类，生成密钥对与管理地址。
- `crypto.py`: 封装 SHA256, RIPEMD160, ECDSA 等加密原语。
- `address.py`: 地址编码与解码工具（Base58）。

### 智能合约与状态
- `contract.py`: 基础智能合约抽象类。
- `vm.py`: 简易虚拟机，用于执行合约字节码。
- `state.py`: 全局状态管理，账户余额映射。

### 存储与工具
- `leveldb_mock.py`: 模拟键值对数据库存储引擎。
- `utils.py`: 通用工具函数（序列化、哈希计算）。
- `logger.py`: 统一日志系统。
- `config.py`: 全局配置中心。

### 接口与测试
- `api.py`: RESTful API 接口，提供链上数据查询。
- `main.py`: 程序入口。
- `test_suite.py`: 单元测试集合。
